- IF-ELSE
> RETURN 後面的語句不會執行
```
IF (CONDITION1) {
  STATEMENT1
} ELSE IF (CONDITION2) {
  STATEMENT2
} ELSE IF (CONDITION3) {
  STATEMENT3
. . .
} ELSE {
  STATEMENTN
}
```

- SWITCH
```
SWITCH (NUM) {
  CASE VALUE1:
    STATEMENT1;
    BREAK;
  CASE VALUE2:
    STATEMENT2;
    BREAK;
...
  CASE VALUEN:
    STATEMENTN;
    BREAK;
}
```

```
SWITCH (NUM) {
  CASE VALUE1:
    STATEMENT1;
    BREAK;
  CASE VALUE2:
    STATEMENT2;
    BREAK;
...
  DEFAULT:
    DEFAULTSTATEMENT;
}
```
