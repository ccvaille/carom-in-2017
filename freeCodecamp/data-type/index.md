## EQUAL
- 相等运算符比较两个值，如果它们是同等，返回 TRUE，如果它们不等，返回 FALSE。
- 值得注意的是相等运算符不同于赋值运算符（=），赋值运算符是把等号右边的值赋给左边的变量。
- === 比较元素的值和 数据类型

```
   1   ==  1    // TRUE
   1   ==  2    // FALSE
   1   == '1'   // TRUE
  "3"  ==  3    // TRUE
   3 === 3   // TRUE
   3 === '3' // FALSE
```
